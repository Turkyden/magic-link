## API

| Params | Description | Default | Required |
| --- | ------ | ----- | --- |
| url | The outer url links of wechat | https://www.baidu.com | true |
| text | The text content displayed in card | 长按识别二维码查看原文 | false |

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```
